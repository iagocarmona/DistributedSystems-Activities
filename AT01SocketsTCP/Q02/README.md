### Como compilar
- Server: 
    - Entre na pasta `server`, executando `cd server`
    - Execute `javac server.java`

- Cliente: 
    - Entre na pasta `client`, executando `cd client`
    - Execute `javac client.java`

### Como executar
- Server: 
    - Entre na pasta `server`, executando `cd server`
    - Execute `java server.java`

- Cliente: 
    - Entre na pasta `client`, executando `cd client`
    - Execute `java client.java`
